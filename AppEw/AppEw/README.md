App de Notícias React Native
Este é um aplicativo simples de notícias desenvolvido em React Native, que permite visualizar uma lista de notícias, adicionar novas notícias e alternar entre modo claro e modo escuro.
Funcionalidades
Visualizar lista de notícias com título, data e imagem.
Alternar entre modo claro e modo escuro usando um switch.
Adicionar novas notícias dinamicamente via input.
Exibir o total de notícias cadastradas.
Tecnologias Utilizadas
React Native
React Hooks (useState)
FlatList para renderização de listas
Switch, TextInput, Button, Image e View para interface
